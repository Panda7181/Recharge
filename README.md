# Jio-style Homepage Clone

This is a static, mobile-first homepage recreated from the screenshot shared by the user.

It visually follows the Jio-style layout in the reference image: blue header, hero banner, mobile recharge card, service shortcuts, content cards, trending devices, MyJio CTA, guidance section, and footer.

The recharge button validates a 10-digit mobile number and opens `plans.html` with demo special-offer plan cards.

## Files

- `index.html` — page structure
- `plans.html` — special offer plans page
- `styles.css` — mobile-first styling
- `script.js` — small tab/recharge demo interactions

## Run

Open `index.html` in a browser.

This is a demo UI only; no real recharge, payment, order, or data collection is connected.
